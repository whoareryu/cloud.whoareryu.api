# 캐글 타이타닉 생존자 예측을 위한 머신러닝 알고리즘 Top 5

캐글(Kaggle) 타이타닉 데이터셋과 같은 정형 데이터(Tabular Data) 분류 문제에서 가장 대중적으로 사용되면서도 높은 정확도를 보여주는 **탑 5 알고리즘** 정리 노트입니다. 타이타닉 데이터셋은 크기가 작으므로, 딥러닝보다는 **트리(Tree) 기반의 앙상블 알고리즘**들이 압도적인 성능을 자랑합니다.

---

## 1. XGBoost (Extreme Gradient Boosting)
현재 캐글러들이 정형 데이터 예측에 가장 애용하는 강력한 부스팅 알고리즘입니다.
* **특징:** Gradient Boosting 알고리즘을 병렬 학습이 가능하도록 최적화하여 속도가 빠르고 성능이 뛰어납니다. 과적합(Overfitting)을 방지하기 위한 자체 규제(Regularization) 기능이 포함되어 있습니다.
* **추천 이유:** 파라미터 튜닝을 조금만 해주어도 기본적으로 80% 이상의 안정적인 상위권 정확도를 보장합니다.

## 2. LightGBM (Light Gradient Boosting Machine)
마이크로소프트에서 개발한 부스팅 알고리즘으로, XGBoost와 양대 산맥을 이룹니다.
* **특징:** 균형 분할(Level-wise) 방식이 아닌 리프 중심 분할(Leaf-wise) 방식을 사용하여 예측 오류 손실을 더 적극적으로 줄입니다. 메모리를 적게 먹고 속도가 압도적으로 빠릅니다.
* **추천 이유:** 타이타닉 데이터가 작아서 체감이 크지 않을 수 있지만, 하이퍼파라미터 튜닝을 세밀하게 할 경우 XGBoost보다 미세하게 더 높은 정확도를 쥐어짜 내기도 합니다.

## 3. CatBoost (Categorical Boosting)
Yandex에서 개발한 알고리즘으로, 타이타닉 데이터셋처럼 **범주형 변수(성별, 승선항, 티켓 등)**가 많은 데이터에 최적화되어 있습니다.
* **특징:** 범주형 데이터를 원-핫 인코딩(One-Hot Encoding)하지 않아도 내부적으로 훌륭하게 처리하며, 과적합을 매우 잘 잡아냅니다.
* **추천 이유:** 기본 설정(Default) 상태에서의 성능이 가장 뛰어납니다. 복잡한 튜닝 없이도 곧바로 최상위권 정확도를 내는 경우가 많아 초보자에게 특히 추천합니다.

## 4. Random Forest (랜덤 포레스트)
머신러닝 분류 문제의 '기본서'와 같은 배깅(Bagging) 기반의 앙상블 알고리즘입니다.
* **특징:** 여러 개의 의사결정 나무(Decision Tree)를 무작위로 만들고, 그들의 예측을 다수결(Soft/Hard Voting)로 종합합니다.
* **추천 이유:** 사용법이 직관적이고 시각화가 가능합니다. 부스팅 모델들에 비해 과적합에 강해서 데이터 전처리가 조금 미흡해도 79~81% 선의 준수한 정확도를 안정적으로 뽑아내 줍니다.

## 5. Voting Classifier (보팅 앙상블)
단일 알고리즘은 아니지만, 캐글 상위권으로 가기 위해 무조건 쓰이는 **'치트키' 기법**입니다.
* **특징:** 위에서 언급한 XGBoost, LightGBM, Random Forest, 혹은 SVM(서포트 벡터 머신) 같은 서로 다른 알고리즘 모델들을 여러 개 만든 뒤, 이 모델들의 예측 결과를 종합하여 최종 투표를 하는 방식입니다.
* **추천 이유:** 각 알고리즘이 가진 약점을 서로 보완해주기 때문에 단일 모델을 사용할 때보다 정확도가 1~2% 정도 확실하게 상승하는 효과를 봅니다.

---

### 💡 핵심 팁: 정확도를 더 올리고 싶다면?
타이타닉 예제에서는 사실 **"어떤 알고리즘을 쓰느냐"보다 "데이터를 어떻게 요리하느냐(피처 엔지니어링)"**가 정확도를 결정짓는 90%의 요인입니다.

1. **파생 변수 생성:** `Name` 열에서 인물들의 호칭(Mr, Miss, Mrs, Master)을 추출하거나, `SibSp`(형제/배우자)와 `Parch`(부모/자식)를 더해 `FamilySize`(가족 수) 생성하기
2. **디테일한 결측치(NaN) 처리:** `Age`나 `Fare`를 단순 전체 평균이 아닌, 호칭별 또는 객실 등급별 중앙값으로 세분화하여 채우기

이러한 전처리를 거친 후에 위의 알고리즘들을 돌려보면 눈에 띄게 높은 점수를 얻을 수 있습니다.
