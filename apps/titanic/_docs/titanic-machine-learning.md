# 타이타닉 생존자 예측 AI 프로젝트 가이드 (프롬프트 피딩용 MD)

이 문서는 타이타닉호 침몰 사고의 탑승객 데이터를 활용하여, 특정 승객의 생존 여부(Survived)를 예측하는 머신러닝 프로세스의 전체 흐름을 정리한 가이드라인입니다. 비전문가의 눈높이에 맞춰 기획된 단계별 구조(데이터 준비 ➔ 정제 ➔ AI 학습 ➔ 결과 제출)와 데이터 척도를 클로드(Claude)가 인식하기 쉽게 포맷팅했습니다.

---

## 1. 데이터 개요 및 준비 (Data Overview & Preparation)

AI를 학습시키기 위해 데이터는 크게 두 가지 그룹으로 나뉘어 처리됩니다.

### ① 학습용 데이터 (`train.csv`)
- **역할:** AI를 훈련시키는 교과서 (기출문제)
- **구성:** 승객들의 개인 정보(성별, 나이, 등급 등 피처)와 함께 실제 생존 여부인 **정답(Ground Truth, `Survived`)**이 포함되어 있습니다.

### ② 시험용 데이터 (`test.csv`)
- **역할:** AI가 얼마나 잘 맞추는지 확인하는 시험지 (실전문제)
- **구성:** 승객들의 개인 정보는 들어있지만, **생존 여부(정답)는 숨겨져 있습니다.** AI 모델은 이 데이터를 바탕으로 예측 결과물을 도출해야 합니다.

### ③ 데이터 측정 척도 분류 (Data Types & Scales)
승객의 정보는 성격에 따라 아래와 같이 4가지 데이터 측정 척도로 분류되어 처리됩니다.

| 데이터 대분류 | 세부 척도 (Type) | 타이타닉 데이터 예시 | 설명 |
| :--- | :--- | :--- | :--- |
| **Categorical**<br>(범주형: 그룹) | **Nominal (명목형)** | 성별(`Sex`), 승선항(`Embarked`), 이름(`Name`) | 순서가 없고 단순히 이름이나 종류를 나타내는 데이터 |
| | **Ordinal (순서형)** | 객실 등급(`Pclass`: 1, 2, 3등석) | 데이터 사이에 명확한 순위나 서열이 있는 데이터 |
| **Quantitative**<br>(수치형: 계산) | **Interval (간격형)** | 본 데이터셋에는 해당 없음 | 절대적 기준점(0)은 없으나 일정한 측정 구간을 가진 데이터 |
| | **Ratio (비율형)** | 나이(`Age`), 요금(`Fare`), 동반자 수(`SibSp`, `Parch`) | 기준점 0이 존재하며, 배수 계산(2배 등)이 가능한 데이터 |
